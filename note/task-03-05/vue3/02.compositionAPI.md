## 设计动机

  * Option API

    - 包含一个描述组件选项（data, methods、 props等) 的对象

    - Options API 开发复杂组件， 同一个功能逻辑的代码被拆分到不同选项
  
  * Composition API

    - Vue.js3.0新增的一组API

    - 一组基于函数的API

    - 可以更灵活的组织组件的逻辑
