# RN
> 混合开发， 引擎jscore

优势：

  * 开发体验好

  * 开发成本低

  * 学习成本低

不足： 

  * 不成熟

  * 性能差： 整体性能不如原生

  * 兼容性差： 涉及底层开发，需要针对andriod 和ios双端单独开发


### 创建项目

* 初始化项目

    **react-native init myproject**

* 进入项目

    **cd ./myproject**

* 运行项目

    **yarn run-andriod**

### 调试

* 点击模拟器
* 快捷键 command + m
* 点击debug
